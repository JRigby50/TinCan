Then in your main HTA I setup a trap for the Click event on the hidden control, something like this ...



<script language=vbs>
Dim oDoc
sub window_onload

  set oDoc = display.document
  set oDoc.all.HiddenTextControl.onclick = getRef("GetData")
end sub
sub GetData()
  sText = oDoc.all.HiddenTextControl.Value
  msgBox "In Main: " & sText
end sub

</script>


Just put the retrieved text into whatever control in the main page that you desire.  (I couldn't find a control named SelectQueue on the page you posted).
